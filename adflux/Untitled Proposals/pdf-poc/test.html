૧. કઠિન જોડાક્ષરો — Conjunct Stress Test

This page deliberately exercises the conjuncts that broke the docx-based pipeline. If any glyph below renders as a dotted-circle placeholder, a square box, or a visible halant (વિરામ), the font fallback is wrong and the proposal output cannot ship.

Word (Gujarati) Phonetic What it tests
ક્ષેત્રે kṣetre The classic ક + ષ ligature. Most weak fonts collapse this to "ક્ ષ".
સ્વચ્છ swacch Double consonant cluster + reph-like joiner; common in govt copy ("Swachh Bharat").
વ્યાજબી vyājabī વ + ય half-form. Failure mode: standalone halant visible.
ર્ડા rḍā Reph-above + matra below — three-stack vertical.
ડાઇરેક્ટર director Loanword with ઇ matra + ક + ટ conjunct + ર.
જાહેરાત jāherāt "Advertisement" — appears on every proposal.
પ્રપોઝલ proposal પ + ર half-form (પ્ર is the most common conjunct in govt prose).
સ્ક્રીન screen Triple-stack at start — સ્ક્ + ર + ી.
ગુજરાત રાજ્ય માર્ગ વાહન વ્યવહાર નિગમ GSRTC full name Long phrase with mixed reph, half-forms, and matras. Tests line-breaking.
